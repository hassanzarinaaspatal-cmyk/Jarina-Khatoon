# Jarina Khatoon

This repository now includes a minimal Expo (React Native) starter app so you can run the project on your mobile device using the Expo Go app.

## What I added
- LICENSE (MIT) — already added
- package.json — dependencies and scripts for Expo
- App.js — minimal React Native app (opens project README)
- .gitignore — to ignore node_modules and typical files

## Run on your mobile (recommended: Expo Go)
1. Install Node.js (v16+ recommended) and npm.
2. On your computer, run:

```bash
npm install
npx expo start
```

3. Install Expo Go on your phone (iOS App Store or Google Play).
4. When `expo start` opens the Metro page in your browser, scan the QR code with Expo Go (Android can scan directly; on iOS, use the Camera app to open the link or copy the tunnel link into Safari).

Alternative: If you prefer native builds, install Android Studio / Xcode and run `npm run android` or `npm run ios`.

## Notes
- This is a minimal starter to get something running on mobile quickly. If you want a full-featured app (offline support, native modules, authentication), tell me and I will add those features.
